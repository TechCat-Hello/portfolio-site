quake_viz（地震可視化アプリ）

概要

USGS（米国地質調査所）のAPIから取得した地震データを日本地図上にマッピングし、
都道府県や年度で絞り込み検索が可能なインタラクティブなDjangoアプリです。

主な使用技術

Python / Django / JavaScript / Leaflet.js / USGS API

主な機能

  • USGS APIを使った地震データの取得・更新
  • Leaflet.jsによる地図上の地震発生地点の可視化
  • マップ上のマーカーをクリックすると、場所・発生時刻・マグニチュードを表示
  • 日付・マグニチュード・都道府県での絞り込み機能
  • 地震発生履歴のCSV・Excel出力機能
  • レスポンシブデザインによるPC・スマホ対応

開発時の工夫と技術的ポイント

  • ユーザー入力（年、都道府県、マグニチュード）に応じて地震データをフィルタリングし、対象地域の情報を抽出
  • 取得した地震データは1ページあたり10件ずつのページネーションに対応し、大量データでも快適に閲覧可能
  • 地図上のマーカークリックで、地震の場所・発生日時・マグニチュードの詳細情報を表示し、インタラクティブ性を向上
  • Djangoのフォームを使った入力データのバリデーションを実装し、エラー時に明確なフィードバックを表示
  • UIにスライダーを導入し、マグニチュードの範囲指定を動的に行い操作性を向上
  • プルダウンでの年度と都道府県の選択により誤入力を防ぎつつ、絞り込み検索を実現
