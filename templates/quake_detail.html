地震可視化アプリ（Earthquake Visualization App）

概要：USGSのAPIから取得したリアルタイムおよび過去の地震データを日本地図上にマッピングし、地域ごとの地震リスクや発生傾向を分析できるインタラクティブなDjangoアプリです。

使用技術：Python / Django / JavaScript / Leaflet.js / USGS API / Plotly

  • USGS APIを使った地震データの取得・更新
  • Leaflet.jsによる地図上の地震発生地点の可視化
  • 日付・マグニチュード・都道府県での絞り込み機能
  • 地震発生履歴のグラフ表示（Plotly使用）
  • レスポンシブデザインによるPC・スマホ対応
