quake_viz（地震可視化アプリ）

概要

USGS（米国地質調査所）のAPIから取得した地震データを日本地図上にマッピングし、
都道府県や年度で絞り込み検索が可能なインタラクティブなDjangoアプリです。

主な使用技術

Python / Django / JavaScript / Leaflet.js / USGS API

主な機能

  • USGS APIを使った地震データの取得・更新
  • Leaflet.jsによる地図上の地震発生地点の可視化
  • マップ上のシンボルをクリックすると、場所・発生時刻・マグニチュードを表示
  • 日付・マグニチュード・都道府県での絞り込み機能
  • 地震発生履歴のCSV・Excel出力機能
  • レスポンシブデザインによるPC・スマホ対応
