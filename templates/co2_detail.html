co2_viz（CO₂排出量可視化アプリ）

概要

World Bank APIから取得した国別のCO₂排出量データを
年・国ごとに絞り込み、表とグラフで可視化するDjangoアプリです。

主な使用技術

Python / Django / Pandas / matplotlib / Bootstrap / World Bank API

主な機能

  • World Bank APIを使ったCO₂排出量データの自動取得
  • 国ごとのCO₂排出量を棒グラフで可視化
  • 国と年のフィルタリング機能
  • CSV・Excel出力機能
  • レスポンシブデザインによるPC・スマホ対応

開発時の工夫と技術的ポイント

  • APIレスポンスの不整合や欠損データに対応し、エラー発生時にメッセージを表示
  • 入力フォームのバリデーションを充実させ、不正な値入力を防止し安定した動作を実現
  • レスポンシブデザイン対応（Bootstrap利用）でスマートフォンからの操作も考慮
  • グラフ描画にMatplotlibを利用し、視覚的にわかりやすいデータ表示を実現
  • データのCSV・Excelダウンロード機能を実装し、ユーザーが外部活用できるように提供
  • グラフは棒グラフ形式を選択し、時系列の変化が一目でわかる視覚表現を意識
