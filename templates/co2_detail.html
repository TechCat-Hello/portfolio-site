CO₂排出量可視化アプリ（Global CO₂ Emissions Visualizer）

概要：国別のCO₂排出量データを可視化するDjangoアプリです。World Bank APIを使用して取得したデータをもとに、インタラクティブなグラフで各国の排出傾向を比較・分析できます。

使用技術：Python / Django / Bootstrap / Chart.js / World Bank API

  • 国ごとのCO₂排出量を可視化（折れ線グラフ）
  • World Bank APIからのデータ自動取得
  • 国と年代でフィルタリング
  • 複数国のデータ比較機能
  • レスポンシブ対応
