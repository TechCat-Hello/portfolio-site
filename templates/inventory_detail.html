Inventory-app（備品管理アプリ）

概要

社内の備品を効率的に管理・貸出・返却・履歴管理できるDjango製の業務支援アプリです。
管理者と一般ユーザーの2種類の権限に対応し、
データ出力・グラフ可視化・PDF出力など、実務に即した機能を備えています。

主な使用技術

Python / Django / Bootstrap / WeasyPrint / Chart.js

主な機能

  • 備品の登録・編集・削除機能
  • 貸出・返却申請の管理および履歴記録
  • ユーザー権限による管理者・一般ユーザーの機能制御
  • カテゴリ別の貸出状況グラフ表示（Chart.js使用）
  • CSV・Excel・PDFでのデータ出力機能
  • レスポンシブデザインによるPC・スマホ対応

開発時の工夫と技術的ポイント

  • ログイン後、ユーザー権限（管理者 / 一般ユーザー）に応じて異なるダッシュボードへ自動リダイレクトし、業務上の役割分担を明確に反映
  • 管理者はすべての備品情報を管理可能、一般ユーザーは自分が登録した備品のみ編集・削除できるように制御（Djangoの認可機構を活用）
  • CSV・Excel・PDF出力機能を実装し、出力フォーマットごとに使いやすさと可読性を意識（PDFはWeasyPrintで日本語対応）
  • 貸出状況をカテゴリ別にグラフ化し、視覚的に在庫管理状況を把握できるUIを構築
  • レスポンシブデザイン対応（Bootstrap利用）でスマートフォンからの操作も考慮
  • 各種フォームでは入力値のバリデーションを行い、エラー時に適切なメッセージを表示
  • Django のテンプレート継承を活用してヘッダーやボタンスタイルを統一し、保守性と見た目の一貫性を両立
