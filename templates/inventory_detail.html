Inventory-app（備品管理アプリ）

概要

社内の備品を効率的に管理・貸出・返却・履歴管理できるDjango製の業務支援アプリです。
管理者と一般ユーザーの2種類の権限に対応し、
データ出力・グラフ可視化・PDF出力など、実務に即した機能を備えています。

主な使用技術

Python / Django / Bootstrap / WeasyPrint / Chart.js

スクリーンショット

一般ユーザーダッシュボード（出力・申請機能）
一般ユーザーダッシュボード（貸出状況グラフ）
